# 2.10 heightsen off

- 설명 
  
    ```heightsen off``` 문은 높이센싱(AVC,Arc 길이제어)을 종료하는 명령문입니다.


- 문법
  
    - ```heightsen off```
      
</br>  

- 사용 예
```python   
   heightsen off            # 높이센싱 실행을 종료
```
